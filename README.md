本系統分為兩大區塊：
左側：輸入區 用於輸入房貸相關資料
右側：結果顯示區 顯示試算結果與每月應繳金額表格（按下計算後才顯示）
輸入項目
房屋總價:輸入房屋總價格（單位：元） 
自備款比例:輸入百分比（不需輸入 % 符號）、0 ~ 100% 
貸款年利率:輸入年利率（百分比）、0 ~ 100% 
貸款年限:貸款年數 
寬限期:使用下拉選單選擇 、不可大於貸款年限 
還款方式選擇 :本息平均攤還法、本金平均攤還法 
操作步驟
1.	輸入所有必要資料 
2.	選擇還款方式 
3.	按下計算按鈕或按 Enter
4.	右側顯示試算結果 
使用注意事項
•	所有輸入需為數字（不含符號） 
•	百分比不需輸入「%」 
系統會在輸入錯誤時顯示紅字提示：
•	自備款比例 > 100% 
•	年利率 > 100% 
•	寬限期 > 貸款年限 
•	非數字輸入 
若有錯誤，系統不會顯示計算結果
